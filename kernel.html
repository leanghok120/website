My progress on sinix - my unix like kernel

I've been programming for quite a while but I never once had a long term side project which I worked on whenever I was bored. I always had a project which I worked on till I considered it completed and then I'd start a new project.

This kernel I'm writing is my first ever long term project which I will come back to work on every once in a while.

So far, the kernel has:

  • a serial driver
  • a PS/2 driver
  • a tiny framebuffer driver where I can draw filled and outlined rectangles
  • pong running on the framebuffer
